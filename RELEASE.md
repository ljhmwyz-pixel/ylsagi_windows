# 发布检查清单

## 必跑验证

```powershell
pnpm install
pnpm run verify
```

## 手动验收

- 首次启动能看到 88x88 悬浮球。
- 点击悬浮球能打开/关闭面板。
- 拖动到屏幕边缘释放后能半隐藏；从半隐藏状态可以拖出。
- 点击面板外能自动关闭；关闭该设置后不自动关闭。
- Token 未配置时显示明确错误，不崩溃。
- Token 有效时今日/本周数据同步更新，悬浮球和面板一致。
- 断网时显示上次成功缓存，并标记缓存状态。
- 托盘菜单能刷新、打开设置、重置位置、打开日志目录。
- 重启应用后悬浮球位置和吸附状态恢复正常。
- 多屏或任务栏位置变化后悬浮球不会跑出屏幕。

## 打包

```powershell
pnpm run pack
```

如果 `electron-builder` 在本机长时间卡住，优先检查网络和 electron-builder 缓存目录。当前代码层验证通过，但最终安装包仍需要在稳定的构建环境里完成一次完整打包验证。
